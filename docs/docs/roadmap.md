# Roadmap

## :lucide-road: Planned

Features etc. we have put on our bucket list, to make the framework more handy and awesome to use.

### :lucide-square-stack: Multiple Inputmodels

At the moment you can have only one input-model for each external task at a time.

We plan to move forward, allowing you to declare multiple input-models. This gives you the possibility to better
structure your inputs and gain better separation within your external tasks code

### :lucide-siren: Metrics

In scalable distributed systems it is crucial to gain insights about operations of your software. Metrics provide
a great way to get a balance between technical and non-technical insights.
