# fastbpmn - camunda7

This example demonstrates how to use fastbpmn together with [camunda7](https://docs.camunda.org/manual/7.24/)

## Prerequisites

* docker installed


## Running the Example

See the [documentation](https://yiogmbh.github.io/fastbpmn/integrations/camunda7/) for
step-by-step instructions on running this example with Docker Compose.


## Remarks

### in-memory database

The example uses an in-memory-database right now, see
[here](https://github.com/cibseven/cibseven-docker?tab=readme-ov-file#configuration-of-the-run-distribution)
how to use a persisted database instead.

### project setup

The example is a single python script file to ensure you can focus on the interaction of fastbpmn and camunda.
