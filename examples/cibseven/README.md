# fastbpmn - cibseven

This example demonstrates how to use fastbpmn togehter with [cibseven](https://cibseven.org)

## Prerequisites

* docker installed

## Remarks

### in-memory database

The example uses an in-memory-database right now, see
[here](https://github.com/cibseven/cibseven-docker?tab=readme-ov-file#configuration-of-the-run-distribution)
how to use a persisted database instead.

### project setup

The example is a single python script file to ensure you can focus on the interaction of fastbpmn and cibseven.
